Cyber Safety Simulator - Run Instructions

1. Prerequisites
   - Python 3.11 or newer
   - Node.js 18 or newer
   - A Google Gemini API key

2. Configure the backend
   - Create a file at: kavach_backend/.env
   - Add the following line:
     GEMINI_API_KEY=your_actual_api_key_here

3. Run the integrated application (recommended)
   - From the project root:
     run_integrated.bat
   - This should build the frontend, copy the built files into the backend static folder, and start the backend server.

4. Manual run steps
   a. Frontend
      cd kavach_frontend
      npm install
      npm run build

   b. Copy built frontend assets into backend static files
      cd ..
      xcopy kavach_frontend\dist kavach_backend\static /E /I /Y

   c. Backend
      cd kavach_backend
      python main.py

5. Open the app
   - In your browser, go to:
     http://localhost:8000
   - Do not use: http://0.0.0.0:8000

6. Notes
   - The backend uses FastAPI and Uvicorn.
   - The frontend is a Vite React app.
   - If you need to run the frontend separately for development, use:
       cd kavach_frontend
       npm run dev

7. Share the app with a client
   - This app runs locally by default. To provide a shareable service link, expose localhost with a tunneling tool.
   - Example using ngrok:
       npm install -g ngrok
       ngrok http 8000
     Then share the generated `https://...ngrok.io` URL with your client.
   - Example using localtunnel:
       npm install -g localtunnel
       lt --port 8000
     Share the generated `https://...loca.lt` URL.
   - For production or MVP deployment, host the backend and built frontend on a public server or cloud service (e.g. Vercel, Render, Cloud Run).
